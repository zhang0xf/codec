# Codec

TLV encoding and decoding implement

## 开发环境

### 1. WSL + vscode (or linux)

使用WSL + vscode搭建开发环境：[WSL + vscode](https://github.com/zhang0xf/md/blob/main/WSL/WSL.md)

### 2. 安装gcc
* `apt-get update`
* `apt-get install gcc`
* `gcc -v`

### 3. 安装make
* `apt-get install make`
* `make -v`

## 编译demo

* `make`
* `make clean`
